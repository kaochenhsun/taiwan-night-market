# 夜遊台灣 Night Market Taiwan

雙語台灣夜市導覽網站，可依縣市、營業日與旅行偏好篩選，並直接開啟地圖導航。

每個夜市都有照片、完整介紹、推薦必吃、交通建議、Google Maps 參考星等及最新評論連結；「幫我選一個」也能直接進入抽中的夜市詳情。

## GitHub Pages 發布

1. 將這個資料夾推送到 GitHub repository。
2. 到 repository 的 **Settings → Pages**。
3. 在 **Build and deployment → Source** 選擇 **GitHub Actions**。
4. 推送到 `main` 後，網站會由內附的 workflow 自動發布。

網站為純 HTML、CSS、JavaScript，不需要安裝套件或建置。

> 夜市營業日可能因天候、節慶或場地安排臨時調整，出發前請再查詢最新公告。

> Google Maps 星等為內容整理時的參考值，實際星等與評論數會持續變動，請以網站內的 Google Maps 連結為準。照片來自 Wikimedia Commons，個別授權與作者標示於詳情頁。
